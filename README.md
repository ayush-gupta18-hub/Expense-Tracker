# Expense Tracker Web Application

## Overview
A client-side expense tracking web application built using HTML, CSS, and JavaScript.  
The application allows users to record daily expenses, categorize them, and view category-wise summaries with persistent storage.

## Features
- Add expenses with title, amount, category, and date
- Client-side input validation and error handling
- Search expenses by title
- Sort expenses by date or amount
- Category-wise expense analytics
- Persistent data storage using browser localStorage
- Delete expenses with confirmation

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Browser LocalStorage

## How to Run
1. Download or clone the repository
2. Open `index.html` in any modern web browser
3. Add expenses using the form interface

## Key Learnings
- DOM manipulation and event handling
- Client-side data validation
- State management using JavaScript arrays and objects
- Implementing analytics through data aggregation
- Persistent storage without backend services
